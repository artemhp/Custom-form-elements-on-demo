Custom Checkbox

Hello, people. I produce you my script of creating custom styled checkboxes. 2012 year overboard, so realization on CSS3 is a good solution. Those people who use old browsers will not appreciate your efforts, I'm shure.
The main goal is to style the way in which the old browsers will show they default elements in right way without deffects.

Style checkbox in form with CSS3

Solution for creating custom checkbox without javascript for this construction of HTML layout:

You can implement custom design to all checkboxes on the page using this CSS stylesheet.

        input[type="checkbox"] + label:before {
            vertical-align: middle;
        }
        input[type="checkbox"] {
            opacity: 0;
            position:absolute;
        }

        label input[type="checkbox"] {
            opacity:1;
            position: relative;
        }


        input[type="checkbox"] + label {
            text-indent:-2px;
            padding-left:20px;
        }

        input[type="checkbox"] + label::before {
            display: inline-block;
            width: 16px;
            height: 15px;

            margin: 0 8px 4px 0;

            line-height: 19px;
            content: "";
            vertical-align: middle;

            background: url('../images/custom_checkbox.png') no-repeat 0 -15px;

        }
        input[type="checkbox"]:disabled + label {
            opacity: .5;
            cursor: default;
        }
        input[type="checkbox"]:not(:checked):not(:disabled):hover + label:before {
            background-position: 0 -30px;
        }
        input[type="checkbox"]:checked + label:before {
            background-position: 0 0;
        }
        input[type="checkbox"]:focus + label {

        }

Looks great, isn't it? But let me show a dark side of this method.

As you know exist another rule of writing construction with label and input: when label wrap input, like this:

In this case our method described upper will not be useful, but even css style is applied to this page inputs will be shown correctly but with no custom styles.

Custom styles for this method we can apply just using some rows jquery or prototype script and modify css stylesheet

Prototype Framework

		$$("input[type=checkbox]").each(function(el){
            if (el.up("label")) {
                el.up("label").addClassName("custom-checkbox-label").down("input[type=checkbox]").insert({
                    after: ""
                });
            }
        });

jQuery

		$$("input[type=checkbox]").each(function(el){
            if (el.up("label")) {
                el.up("label").addClassName("custom-checkbox-label").down("input[type=checkbox]").insert({
                    after: ""
                });
            }
        });

Use only one script on jquery or prototype.
